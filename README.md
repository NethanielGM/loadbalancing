## About The Project

This app balances the load on a server and displays the container IP. Using: ReactJS, NodeJS, Docker, Nginx. Deployed With S3 And EC2

### Built With

- #### ReactJS
- #### NodeJS
- #### ExpressJS

### Deployed With

- #### S3
- #### EC2

## Available Scripts

### To run client

### `npm start`

### To run server

### `docker-compose up --scale api={$NUMBER OF APPS YOU WANT}`
